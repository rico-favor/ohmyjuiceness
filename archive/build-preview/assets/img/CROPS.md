# Image pipeline — crop, grade, derivative log

Generated by `build/scripts/process-preview-images.sh`. All raw sources are
portrait in their true (EXIF-corrected) orientation — several carry an
EXIF `Orientation` tag that rotates a landscape-stored sensor capture to
portrait (4284×5712 / 5712×4284 pairs), so every crop below starts from the
`-auto-orient`-corrected 4284×5712 frame, not the raw file dimensions.

Grade variants:
- **standard** — daylight shots: `-linear-stretch 0.3%x0.3% -auto-gamma`,
  then `-modulate 102,114,100 -sigmoidal-contrast 2.5x50% -fill '#FF8E06' -colorize 3%`.
  (The brief's literal `-tint 3` blew out to a posterized near-black/neon
  result on every test image — verified step-by-step; `-colorize 3%` gives
  the intended subtle brand-warm wash without the artifact. Documented as a
  deliberate deviation.)
- **lowlight** — evening/mixed-artificial-light interiors + Eastwood, no
  tint: `-linear-stretch 0.3%x0.3% -auto-gamma`, then
  `-modulate 102,110,99 -enhance`.
- **graphic** — the flat vector machine-wrap render (machine-vendo-1): only
  `-linear-stretch 0.3%x0.3% -auto-gamma`, no modulate/tint/enhance, to keep
  the brand hex (#FF8E06 etc.) exact.

## Contact sheet QA

`contact-sheet.jpg` (5-wide montage of all 23 primary outputs, gitignored
from the commit — not a deliverable) was viewed after each grading pass.
**First pass** revealed two real bugs, both fixed and re-verified before
final approval:
1. The brief's literal `-fill '#FF8E06' -tint 3` recipe produced a
   posterized/near-black neon result on every "standard"-graded image —
   replaced with `-colorize 3%` (see grade variants above).
2. `machine-parqal.jpg` / `machine-parqal-2.jpg` rendered sideways in the
   montage — their EXIF Orientation=6 tag was correct/real (not stale) and
   the pixel data needed physical rotation; fixed by adding `-auto-orient`
   before grading those two specific already-processed sources.

**Final pass**: all 23 tiles read upright, grade is visually consistent
(warm brand-orange across daylight and lowlight variants, no fake/neon
artifacts, no clipped shadows), no heads/hands cut at frame edges, no
Eastwood upscale mush. Approved.
